Grand Theft Auto San Andreas Widescreen Fix

Supported exe version: v1.0 US
Installation:

  1. Download and extract the archive to the game directory, where the exe is located.
  2. Download and install Widescreen HOR+ by Wesser (required).
  3. Check the ini file for available features.

Notes:

  • A recompiled wshps.asi (Widescreen HOR+) is available if the original does not work.
