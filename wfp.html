Grand Theft Auto Vice City Widescreen Fix

http://www.rockstargames.com/vicecity/

Supported exe version: Any | v1.0 recommended

Installation: Unpack content of archive to GTA VC root directory. Also download archive of widescreen frontend textures(recommended, but optional). You may use content of scripts folder without d3d8.dll if that's necessary(copy to mss or root dir). d3d8.dll also contains d3d8to9 wrapper activated by default. It is used for mods that require rwd3d9.dll, like transparent menu (available in dropdown menu). Check the .ini file to get more info about available features.

Updater for Widescreen Fixes

modupdater is a tool that allows to automatically update outdated plugins (and other files).


Installation:
Put modupdater.asi in the folder with widescreen fix. It's already included with all archives on this page, no need to download it separately anymore.

With "Update all downloaded files" checkbox, updater will replace all downloaded files from the archive. That means, in case ini file is in the archive, it will be replaced and you'll have to configure it again if necessary. If this checkbox is not set, only one file will be updated from that downloaded archive (.asi by default).
